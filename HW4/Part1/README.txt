# COMP 4610 - Homework 4 Part 1

## Dynamic Multiplication Table Using jQuery Validation

## Project Description

This project creates a dynamic multiplication table based on user input. The user enters minimum and maximum values for rows and columns, and the table is generated automatically after successful validation.

The project uses JavaScript, jQuery, and the jQuery Validation Plugin to ensure that all user inputs are valid before the table is displayed.

---

## Features

* Dynamic multiplication table generation.
* Input validation using the jQuery Validation Plugin.
* Custom validation rule to ensure minimum values are less than or equal to maximum values.
* Accepts values between -50 and 50.
* Displays meaningful error messages for invalid input.
* Generates the multiplication table without reloading the page.

---

## Validation Rules

The following rules are enforced:

1. All fields are required.
2. Inputs must be numeric values.
3. Values must be between -50 and 50.
4. Minimum Column Value must be less than or equal to Maximum Column Value.
5. Minimum Row Value must be less than or equal to Maximum Row Value.
